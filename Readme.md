# contao_hjk_vb_phoenix
Modul für das CMS Contao (https://contao.org) zum Einbetten von Tabellen, Spielplänen und Ergebnislisten des Volleyball-Ligabetriebs in Nordrhein-Westfalen / Deutschland.
